# markdown 教程

## 简介



学生时代我们是怎么学习的？都会有课本和板书，还有笔记。课本知识全面详细不清晰、重点不突出。板书相当于老师的笔记，但是我们的笔记可以来帮助我们梳理知识点，更加满足我们的需求，提高学习效率。



markdown 轻量级标记语言 工具使用Typora 文本编辑器



## 笔记的正确**定位**



+ 条理性
+ 符合个人需求
+ 有复习价值即可



## 使用（语法）说明



### 1. 多级标题

```md
# 一级
## 二级
### 三级
#### 四级
##### 五级
###### 六级
```

​      快捷键 

- ctrl+0 清空 标题格式
- ctrl1--ctrl6 一级到6级标题
